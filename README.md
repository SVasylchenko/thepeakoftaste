# ThePeakOfTaste Starter Site

Starter static website for GitHub Pages.

## Files
- `index.html` — main page
- `style.css` — styles
- `script.js` — mobile menu
- `images/` — place your real bakery photos here

## GitHub Pages setup
1. Create a public repository named `thepeakoftaste`
2. Upload all files from this folder
3. In GitHub: Settings → Pages
4. Set:
   - Source: Deploy from a branch
   - Branch: `main`
   - Folder: `/(root)`
5. Save and wait for publication

## Before publishing
Replace placeholders:
- email
- phone
- Instagram link
- menu items
- bakery story
- gallery photos
